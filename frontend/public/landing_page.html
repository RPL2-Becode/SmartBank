Fitur SmartBank

Semua kebutuhan banking UMKM Anda dalam satu platform yang lengkap dan aman.

👤

Registrasi & Login

Daftar dengan mudah menggunakan User ID. Login aman dengan JWT token dan bcrypt password hashing.

💰

Cek Saldo

Pantau saldo terbaru secara real-time dengan tampilan yang clean dan informatif.

💸

Transfer

Transfer saldo ke pengguna lain dengan cepat dan aman. Tersedia berbagai metode transfer.

📊

Pinjaman (Loan)

Minta pinjaman hingga Rp 100 juta dengan bunga 10% per tahun. Tersedia 3 tier: Reguler, Gold, Prioritas.

📜

History Transaksi

Lihat semua riwayat transaksi dan ledger 100 transaksi terakhir dengan detail lengkap.

🔐

Keamanan Tinggi

SQL Injection prevention, XSS protection, Rate limiting, dan Helmet.js security headers.

Keamanan Terbaik

SmartBank dilengkapi dengan fitur keamanan tingkat enterprise untuk melindungi data Anda.

🔑

JWT Authentication

Autentikasi dengan JWT (HS256) dengan expire 1 hari dan tamper detection untuk keamanan maksimal.

🛡️

SQL Injection Prevention

Parameterized queries menggunakan mysql2 untuk mencegah SQL injection attacks.

⚔️

XSS Protection

Helmet.js headers dan Zod validation untuk melindungi dari cross-site scripting.

🚦

Rate Limiting

Batasi request: 5 req/menit untuk auth, 50 req/menit untuk endpoint umum.

🔒

bcrypt Password Hashing

Password di-hash dengan bcryptjs (10 rounds) sebelum disimpan ke database.
