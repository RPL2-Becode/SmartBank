Keamanan Tingkat Bank

Standar keamanan tertinggi untuk melindungi data dan aset Anda

SmartBank menggunakan infrastruktur keamanan kelas enterprise dengan multi-layer protection untuk memastikan setiap transaksi Anda aman dan terlindungi.

JWT Authentication

Token-based authentication dengan enkripsi HS256 untuk keamanan akses yang tidak dapat di-tamper.

POST /smartbank/auth/register No Auth

Daftar akun user baru dengan role NASABAH.

{ "userId": "userman", "name": "User Manual", "password": "Password123!", "role": "NASABAH" } // Response: 201 Created
POST /smartbank/auth/login No Auth

Login user dan mendapatkan JWT token untuk autentikasi.

{ "userId": "userman", "password": "Password123!" } // Response: 200 OK // Returns: { token, user }
GET /smartbank/balance JWT Required

Cek saldo dan history transaksi user yang login.

// Header: Authorization: Bearer [token] // Response: 200 OK { "status": "success", "data": { "balance": 50000.00, "loan": 0.00, "history": [...] } }
POST /smartbank/transfer JWT Required

Transfer saldo ke user lain dengan biaya admin dan pajak.

{ "toUserId": "tujuan", "amount": 50000 } // Biaya: FEE_BANK (1%) + TAX_RATE (2%) // Max 10 transaksi/hari, cooldown 10 detik
POST /smartbank/loan JWT Required

Ajukan pinjaman dengan bunga 10% per periode (30 hari).

{ "amount": 1000000 } // Response: { "loanId": 1, "amount": 1000000, "interestRate": 0.10, "totalDue": 1100000, "status": "APPROVED" }
GET /smartbank/ledger JWT Required

Ambil ledger 100 transaksi terakhir untuk audit trail.

// Header: Authorization: Bearer [token] // Response: Array of transactions // Fields: refId, type, fromUserId, toUserId, amount, tax, fee, description, created_at
